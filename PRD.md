3. **AR手部打地鼠** - 使用手部追蹤技術！伸出手指對準鏡頭，用食指打地鼠。需要相機權限和良好的光線環境。

## 功能需求

### 核心功能
1. 遊戲中心首頁展示所有遊戲
2. 點擊遊戲卡片進入對應遊戲頁面
3. 支援黑暗/明亮模式切換
4. 遊戲數據統計顯示（最高分、遊玩次數等）
5. 響應式設計，支援手機、平板、電腦

### 技術需求
1. 使用 HTML5、CSS3、JavaScript（ES6+）
2. 無需後端，純前端實現
3. 使用 localStorage 保存遊戲數據
4. 支援現代瀏覽器（Chrome、Firefox、Safari、Edge）
5. 遊戲3需要相機權限和MediaPipe Hands庫

### 設計需求
1. 繁體中文介面
2. 黑暗模式與明亮模式切換
3. 遊戲卡片懸停效果
4. 流暢的動畫過渡
5. 直觀的用戶界面
