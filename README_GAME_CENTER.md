# Helena's Game Center - 遊戲中心

## 項目概述

這是一個結合明亮/黑暗模式的遊戲中心主頁，整合了三個現有的HTML遊戲。主頁提供了現代化的用戶界面，具有模式切換功能、響應式設計和直觀的遊戲導航。

## 檔案結構

```
遊戲中心/
├── index.html              # 主頁檔案（新創建）
├── game1.html              # 2048經典版遊戲
├── game2.html              # 3D森林打獵遊戲
├── game3.html              # AR手部打地鼠遊戲
├── README_GAME_CENTER.md   # 項目文檔
└── plans/                  # 計劃文件
    ├── homepage_integration_plan.md
    └── game_center_homepage_implementation.md
```

## 功能特性

### 1. 模式切換功能
- **明亮模式**：適合白天或明亮環境使用
- **黑暗模式**：適合夜間或暗環境使用
- **自動檢測**：根據系統偏好自動選擇模式
- **本地存儲**：記住用戶的偏好設置

### 2. 遊戲整合
- **2048經典版** (`game1.html`)：數字合成益智遊戲
- **森林打獵** (`game2.html`)：3D射擊遊戲
- **AR打地鼠** (`game3.html`)：手部追蹤AR遊戲

### 3. 響應式設計
- **桌面端**：三列卡片佈局
- **平板端**：雙列卡片佈局
- **移動端**：單列卡片佈局
- **自適應字體**：根據屏幕尺寸調整

### 4. 用戶體驗
- **懸浮效果**：卡片懸浮動畫
- **玻璃態效果**：半透明背景
- **漸變設計**：現代化漸變色彩
- **平滑過渡**：所有狀態變化都有動畫

## 技術棧

- **HTML5**：語義化標記
- **Tailwind CSS**：實用優先的CSS框架
- **JavaScript**：交互功能
- **Google Fonts**：Spline Sans字體
- **Material Icons**：Material Design圖標

## 使用方法

### 本地運行
1. 確保所有檔案在同一目錄下
2. 使用任何現代瀏覽器打開 `index.html`
3. 或者使用本地服務器（如Live Server）

### 功能操作
1. **切換模式**：點擊右上角的模式切換按鈕
2. **開始遊戲**：點擊任何遊戲卡片的"立即開始"按鈕
3. **查看排行榜**：點擊歡迎區的"查看排行榜"按鈕
4. **遊戲指南**：點擊"遊戲指南"按鈕查看遊戲說明

## 遊戲詳細信息

### 1. 經典 2048 (`game1.html`)
- **類型**：益智遊戲
- **控制方式**：鍵盤方向鍵或WASD
- **目標**：合成2048方塊
- **特色**：本地存儲最高分、響應式設計

### 2. 森林狩獵 (`game2.html`)
- **類型**：3D射擊遊戲
- **控制方式**：滑鼠瞄準和點擊
- **目標**：射擊動物獲得高分
- **特色**：Three.js 3D渲染、計時器系統

### 3. AR手部打地鼠 (`game3.html`)
- **類型**：AR互動遊戲
- **控制方式**：手部追蹤（需要攝像頭）
- **目標**：用手部打擊地鼠
- **特色**：MediaPipe手部檢測、AR效果

## 開發指南

### 自定義主題
要修改主題顏色，編輯 `index.html` 中的Tailwind配置：

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: {
                    DEFAULT: '#6c2bee', // 主要顏色
                    light: '#8a5cf0',
                    dark: '#4d1ca8',
                },
                // ... 其他顏色
            }
        }
    }
}
```

### 添加新遊戲
要添加新遊戲到主頁：

1. 在遊戲卡片區添加新的卡片HTML結構
2. 確保連結指向正確的遊戲檔案
3. 為新遊戲選擇合適的顏色和圖標

### 響應式調整
主頁使用Tailwind的響應式類：
- `grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3`
- 移動端：單列
- 平板端：雙列
- 桌面端：三列

## 故障排除

### 常見問題
1. **模式切換不工作**
   - 檢查瀏覽器控制台是否有錯誤
   - 確保JavaScript已正確加載

2. **遊戲連結無法打開**
   - 確認遊戲檔案存在於同一目錄
   - 檢查檔案路徑是否正確

3. **響應式設計問題**
   - 確保viewport meta標籤正確
   - 檢查CSS媒體查詢

4. **圖片無法加載**
   - 檢查網路連接
   - 確認圖片URL有效

### 瀏覽器兼容性
- 需要現代瀏覽器支持ES6+
- 需要支持CSS Grid和Flexbox
- 黑暗模式需要CSS自定義屬性支持
